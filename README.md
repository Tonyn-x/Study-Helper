# 📚 Sistema de Controle de Estudos

Projeto prático desenvolvido utilizando HTML, CSS e JavaScript para o curso de Análise e Desenvolvimento de Sistemas da Uninter.

🔗 **Link do projeto online:** https://tonyn-x.github.io/Study-Helper/

---

## 🎯 Visão Geral do Sistema
O Sistema de Controle de Estudos é uma aplicação web desenvolvida para auxiliar estudantes no gerenciamento de seu tempo. O sistema permite o cadastro de disciplinas, registro de carga horária total e o acompanhamento das horas já estudadas, gerando automaticamente o percentual de progresso do aluno.

## 💻 Tecnologias Utilizadas
- **HTML5:** Estruturação semântica da página.
- **CSS3:** Estilização responsiva (adaptável para computadores e celulares) utilizando Flexbox e CSS Grid.
- **JavaScript:** Criação de toda a lógica de programação, cálculos matemáticos e manipulação do DOM.
- **LocalStorage:** API do navegador utilizada para persistência de dados.

## ⚙️ Funcionalidades (Requisitos Atendidos)
- **Cadastro:** Inserção de novas disciplinas contendo Nome, Professor, Carga Horária e Área do Conhecimento.
- **Listagem e Resumo:** Exibição dinâmica em cards e um painel global que soma todas as horas estudadas.
- **Atualização (Edição e Adição de Horas):** Possibilidade de somar novas horas de estudo à disciplina e editar os dados cadastrais.
- **Exclusão:** Remoção de disciplinas da lista e do banco de dados local com confirmação de segurança.
- **Validações:** O sistema bloqueia a inserção de cargas horárias em branco, negativas ou que contenham texto.

## 🚀 Como Executar
1. Clone este repositório ou baixe o arquivo ZIP.
2. Extraia os arquivos.
3. Abra o arquivo `index.html` em qualquer navegador web.
4. Opcional: Acesse diretamente pelo link do GitHub Pages disponibilizado no topo desta documentação.
